What is a Way-point?

This tool allows the user to define an observation by generating a path (of a defined width/offset value) and captures an area along a linear corridor. This is usually used to mark observations of invasive species that occur along roads and rivers. It can be used to properly define the area spatially.
The interface allows the user to enter points based on their UTM location i.e. Easting and Northing values.
Upon completion, the user clicks to proceed and generate a map. A buffer is created and displayed on a map.

Please note:

- Points must be sequential to generate a map that is exactly as recorded on ground.

- Minimum of 2 points are required to create a linear way-point path.

- Maximum of 20 points can be recorded.

- The width or offset value is recorded meters. It is applied to either side of the linear path. (e.g. offset = 2 meters , total width = 4 meters.)

- Offset value can range between 1 meter to 20 metres.

- Maximum distance between 2 points is 200 meters.

- Minimum distance between 2 points is 1 meter.

- Maximum length of the entire plot is 3800 meters.

- Minimum length of the entire plot is 1 meter.
